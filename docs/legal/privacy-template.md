# Privacy Policy Template

This is a starter template, not legal advice.

## Data Collected

- Account profile data.
- Search target metadata.
- Normalized findings from enabled sources.
- Export and audit events.

## Data Not Collected

- Raw credentials.
- Session cookies.
- Private keys.
- Provider secrets.

## Retention

Set retention by organization policy. Default recommendation: 30 days for search results, 1 year for audit events, shorter retention for educational deployments.

## User Rights

Provide deletion, export, correction, and abuse reporting workflows before public launch.
